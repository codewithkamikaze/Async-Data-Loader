# Async Data Loader

A lightweight frontend project built with vanilla JavaScript that demonstrates asynchronous data fetching using the Fetch API.

## 🚀 Features

- Fetches user data from a public API
- Loading spinner indicator
- Error handling with user-friendly messages
- Retry functionality
- Toast notifications
- Responsive grid layout
- Clean modern UI

## 🛠 Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Fetch API
- Async/Await

## 📡 API Used

https://jsonplaceholder.typicode.com/users

## 📷 Preview

Loads users from a public API and displays:
- Name
- Username
- Email

## 💡 What This Project Demonstrates

- Handling asynchronous operations
- Managing UI states (loading, success, error)
- DOM manipulation
- Clean separation of UI logic
- Basic UX improvements

## ▶ How to Run

1. Clone the repository
2. Open `index.html` in your browser

No installation required.
